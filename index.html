Javascript Function Scope

This is a little application ot demonstrate function scope in javascript. A good way to do visualize it is as a landscape defined by the left-hand indentation level of the code. With bridges over every valley

To find out what variables are in scope on a given line move uphill. From that line simultaneously in both direction. Never go downhill (take the bridge over the valley). Every variable and parameter declaration that you walk past will be in scope. Once you get to the top add in all properties of the global window object, this will be everything in your current scope!

Click the next scope button to highlight the next scope up from the selected line. This will work with any code in the below <pre> block. To change the start line or the columns to highlight, modify the variables at the top of the coffeescript tab.

    var informal = "Hi";
    
    var sayHi = function() {
        var formal = "Hello";
